4.  Login no apricativo:
        Url+'/alunos/getAluno/login=<usuário>&senha=<senha>'
    Respostas esperadas
    sucesso:
        {   "usrNome":  <nome>,
            "usrMat":   <matrícula>,
            "token":    <token>
        }
    falha:
        Erro 483 (atualmente qualquer erro retornado pelo servidor nessa url é tratado como usuário não autenticado.)

5.  Cadastro de usuário:
        Url+'/alunos/setAluno/'
    Dados enviados via POST:
        {       nome:      <nome completo>,
                matricula: <matrícula>,
                login: {
                  accessKey: <nome>,
                  password:  <senha codificada>
                }
        }
    Respostas esperadas:
    sucesso:
        200 OK
    conta existente:
        Erro 483 conta já existe
